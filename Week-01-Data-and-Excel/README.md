## Overview
In Week 1, I explored the fundamentals of data analytics and gained practical experience using Excel. The focus was on understanding the importance of data analysis, core data concepts, and the tools and techniques used to process and analyse data effectively.

---

## Key Learnings

### Data Analytics Fundamentals
- Importance of analysing data to make informed decisions
- Key data concepts and terminology
- Understanding relevant laws and regulations when working with data
- Types of data analytics:
  - Descriptive
  - Diagnostic
  - Predictive
  - Prescriptive
- Structured vs. unstructured data
- Quantitative vs. qualitative data
- Data collection and storage methods
- Basic statistical fundamentals
- Privacy and security considerations

### Excel Fundamentals and Advanced Features
- Basic features and functions:
  - Sorting and filtering data
  - Changing data types
  - Implementing formulas and functions (SUM, AVERAGE, VLOOKUP, IF)
  - Validating, filtering, and importing data to support analysis
- Advanced features:
  - Pivot tables and slicers
  - Macros and switch statements
  - Data visualisations 

---

## Skills & Knowledge Developed
- Applying Excel for both basic and advanced data analysis tasks
- Understanding the data analytics lifecycle from collection to visualisation
- Interpreting and preparing data for analysis
- Ensuring data accuracy, security, and compliance
